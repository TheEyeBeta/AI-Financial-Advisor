# SQL Migrations

This directory contains SQL scripts for managing the database schema. Migrations should be applied in chronological order.

## Applying Migrations

To apply a new migration, run the script against your Supabase database. You can do this through the Supabase SQL editor or by connecting to the database directly.

For example, to apply the `add-onboarding-columns.sql` migration, copy the contents of the file and execute it in the Supabase SQL editor.
